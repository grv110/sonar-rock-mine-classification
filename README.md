# 🚀 Sonar Rock vs Mine Classification

This project uses Machine Learning to classify sonar signals as **Rock (R)** or **Mine (M)**.

---

## 📊 Dataset
- 208 samples
- 60 numerical features
- Labels:
  - Rock (R)
  - Mine (M)

---

## ⚙️ Workflow
1. Data Collection & Loading
2. Data Preprocessing
3. Feature & Label Separation
4. Train-Test Split (90-10)
5. Model Training (Logistic Regression)
6. Model Evaluation
7. Prediction System

---

## 🤖 Model Used
- Logistic Regression (Binary Classification)

---

## 📈 Performance
- Training Accuracy: ~83%
- Testing Accuracy: ~76%

---

## 🔮 Prediction
The model predicts whether the object is:
- Rock 🪨
- Mine 💣
